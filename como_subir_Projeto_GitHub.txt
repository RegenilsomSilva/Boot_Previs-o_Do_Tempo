

1° passo  = Abri uma pasta e dentro da pasta abri o Git Brach here
2° passo  = Comando Git innit 
3° passo  = Git status
4° passo = Git add . ("Adcionar )
-----  Versão -- Commit
5° passo  = git commit -m ->Mensagem e com "coloque a versão do seu codígo aqui "

--------Enviar os arquivos Versionado para Nuvem-------------
6° passo  ==  git push 

7° passo Ir no Github e Criar um  Novo repositório
8° passo pega o link que foi criado o Novo repositório  = (git remote add origin e cole o link)
9° passo    git push = Vai da o Erro 
10° passo git push --set-upstream origin master (enter)

==============================
Como Criar uma Nova breanch de Desenvolvimento ?

11° passo =  Git branch 
12° passo = git branch (Nome da nova brench , Desenvolvimento)
13° passo = git checkout -> Muda de brench 
14° passo = git checkout Desenvolvimento
